CozyCoffee

Soha Sarvath Abdul Azeem and Rimsha Sarvath Abdul Azeem

Project Description:

CozyCoffee is an online cafe designed in a warm and inviting cafe style. The website was created using HTML, CSS, JavaScript and PHP, giving it a cozy, user-friendly interface for all users. You will find a full-featured web application that combines both front end design and back end functionality to provide visitors with an easy way to browse the cafe and view its menu while enjoying the overall experience.

Required Environment:

To run CozyCoffee locally, you need:

Web Browser: Chrome, Firefox, Edge, or Safari

Web Server with PHP support: XAMPP, WAMP, MAMP, or any server that supports PHP

Development Environments:

NetBeans IDE for PHP
  
Phoenix Code for HTML, CSS, and JavaScript
